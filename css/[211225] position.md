# Position

## position: static

- 모든 요소의 기본 position 값

## position: relative

- 다른 요소들에게 영향을 주지 않음
- 자기 자신의 본래 위치가 기준

## position: absolute

1. display: block으로 자동으로 변경
2. 길막을 하지 못함(margin이 없다)
3. 집 나간 내 새끼, 찾을 길 없네
4. 기준점: position: static 요소를 제외한 상위 요소를 기준

## position: fixed

1. 기준점: viewport (브라우저 창의 전체 크기)
